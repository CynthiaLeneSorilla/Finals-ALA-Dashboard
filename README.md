# 🌍 SDG 13: Climate Action Dashboard

## 📌 Project Overview
This project analyzes global CO₂ emissions and identifies key drivers affecting climate change indicators across countries from 2000 to 2020.

It uses real-world datasets from the World Bank and applies data visualization techniques to support Sustainable Development Goal 13: Climate Action.

---

## 🎯 Research Question
What factors influence CO₂ emissions across countries over time?

---

## 📊 Key Indicators Analyzed
- CO₂ Emissions
- GDP per Capita
- Urban Population
- Total Population
- Electricity Access

---

## 🧠 Methods Used
- Data Cleaning and Transformation
- Data Merging across multiple datasets
- Feature Engineering (GDP per capita, log CO₂)
- Descriptive Analytics
- Correlation Analysis
- Interactive Visualization

---

## 📈 Dashboard Features
- Year-based filtering
- KPI summary cards
- Global CO₂ choropleth map
- CO₂ trend analysis over time
- Top emitting countries
- Driver importance visualization
- Country-level time series explorer

---

## 🛠️ Tools Used
- Streamlit
- Pandas
- NumPy
- Plotly
